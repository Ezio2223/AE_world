This website was meant to be a birthday gift to my beautiful girl-friend Annie, but I underestimated the amount of work building a website requires, making it a a-month-late birthday gift.

This website is 100% front-ended with my limited knowledge and asset. You will find some solutions and techniques towards some of the problem I met during programing if you`re interested.


How did I build this within 3 month?

As we all know (or at least should know), front-end programing requires 3 different programming language, HTML, CSS, And JS(javascript), it is extremely hard for a beginner like me to build a website all by myself. So it is no shame to admmit that I used another alredy developed wibeste for reference.

All you need is a browser like chrome and a notebook like sublime-text. You can press "inspect" to check a well-developed website to find which and where you can replace with your elements.

Thanks to the great platform of github, for make it possible for everyone to build websites for free, all you have to do is go to github and start your repositery.
